Koda User Guide

Push-to-talk voice transcription for Windows
Version 4.2.0

What is Koda?

Koda is a push-to-talk voice transcription tool for Windows. Press a hotkey, speak, and your words are instantly typed into whatever you're working in — emails, Slack, Word, Excel, ChatGPT, anything.

No clicking a mic button. No switching apps. Just talk.

Installation

1
Double-click KodaSetup-4.2.0.exe
2
If Windows shows a "protected your PC" warning, click More info → Run anyway
3
Follow the installer wizard — takes about 2 minutes
4
Koda launches automatically when done
Look for the tray icon — Koda appears in the bottom-right corner of your screen near the clock. That's how you know it's running.

Basic Usage

Action
What to do
Start recording
Hold Ctrl + Space
Speak
Talk naturally while holding the keys
Paste transcription
Release Ctrl + Space

Your words appear in whatever app or text field was active when you pressed the hotkey.

Tips: Speak at a normal pace. Works in emails, Slack, Word, Excel, browsers, and chat apps. A quiet room gives better accuracy than an expensive microphone.

Microphone Guide

You don't need an expensive microphone. Here's what to expect from each type:

Microphone Quality Notes
Built-in laptop mic Good Works well in a quiet room
USB headset / earbuds Better Recommended — affordable at $15–40
Dedicated USB mic Best Blue Yeti, HyperX — $60–100
Biggest factor: A quiet environment matters more than mic quality. Background noise reduces accuracy more than anything else.
Make sure your mic is set as the default recording device in Windows:
Right-click the speaker icon → Sound settings → Choose your input device

Formula Mode (Excel & Google Sheets)

Koda can convert spoken descriptions into Excel / Google Sheets formulas. Enable it in Settings → Formula mode.

When formula mode is on and you're in Excel or Google Sheets, speak naturally and Koda types the formula:

What you say What Koda types
sum B2 to B10=SUM(B2:B10)
average of A1 to A20=AVERAGE(A1:A20)
max of C1 to C20=MAX(C1:C20)
today=TODAY()
if A1 is greater than 10 then yes else no=IF(A1>10,"yes","no")
vlookup A1 in B1 to D10 column 2=VLOOKUP(A1,B1:D10,2,0)
count B2 to B10=COUNT(B2:B10)
No match? If your phrase doesn't match a formula pattern, Koda pastes it as normal text — so regular dictation in Excel still works.
Complex formulas: Install Ollama (free, local AI) and enable LLM Polish in Settings. Run ollama pull phi3:mini once to set it up.

Troubleshooting

Koda isn't pasting anything
Click into the text field first, then press the hotkey. Check the tray icon is showing (not paused).
Hotkeys stopped working
Right-click the tray icon → Quit, then relaunch Koda. This is rare — Koda recovers from most hotkey issues automatically.
Transcription is inaccurate
Try the Accurate model in Settings (slower but more precise). Reduce background noise. Speak closer to the mic.
No microphone detected
Plug in your mic before launching Koda, then set it as the default recording device in Windows Sound Settings.

Uninstalling

Go to Windows Settings → Apps → Installed apps, find Koda, and click Uninstall.
